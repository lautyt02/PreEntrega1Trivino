PreEntrega1Triviño

Descripción

Programa que genera un arreglo con las caracteristicas seleccionadas por el usuario y le permite realizar varias operaciones con él,tales como imprimir los elementos por posición ascendente y descendente, mostrar el mayor y menor valor valor y calcular el promedio.

Instrucciones

  1. Abra la consola del navegador
  2. Ingrese la cantidad de elementos de su arreglo
  3. Dele valores a los elementos del arreglo
  4. seleccione del menú la operación que desee realizar.
